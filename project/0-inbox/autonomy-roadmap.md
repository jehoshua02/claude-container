# Autonomy roadmap

## Abstract

Strategize the path to safe autonomous task completion. Define capabilities, guardrails, human/AI boundary, and efficient use of Claude limits.

## Details

- What tasks should the agent be able to complete without human input?
- What tasks should always require human approval?
- What guardrails prevent damage (resource limits, egress control, rollback)?
- What monitoring/alerting is needed to trust it?
- How to use Claude API limits efficiently — avoid waste, idle time, and runaway token spend.
- What's the incremental path — what do we unlock first?
