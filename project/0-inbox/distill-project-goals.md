# Distill goals of this project

Define the north star and concrete goals for claude-container. What are we building and why? What does success look like?

## Notes

- claude-container is a platform, not an application. It provides tools, plugins, security, and extensibility.
- User defines the agent's goals and workflows. claude-container just makes sure everything needed for full automation is there.
- Single-agent-excellence tasks (autonomous execution, self-deploy, etc.) are one use case, not claude-container's scope.
