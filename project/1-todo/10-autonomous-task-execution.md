# Autonomous task execution

## Abstract

Agent runs a task from start to finish unsupervised. Given a task file, agent picks it up, completes it, proves it works, and moves it to done. Foundation for everything else.

## Priority: 10

- Value: 4 — foundation for everything else, high unblocking potential
- Momentum: 1 — nothing started
- Effort: 4 — significant scope: pick up task, implement, verify, move to done, all without human input
- Risk: 3 — unsupervised execution has potential to break things or go off-rails
